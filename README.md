# Экспериментальная часть ВКР: Мультибазовая система для интеграции данных о здоровье

## Описание
Данный репозиторий содержит код и результаты экспериментов для магистерской ВКР по теме «Построение мультибазовой системы, интегрирующей информацию о показателях здоровья пользователей тренажёрных залов». Эксперименты демонстрируют работоспособность предложенной архитектуры медиатора на синтетических данных.

## Содержимое
- `experiment.py` – основной скрипт на Python (генерация данных, медиатор, запросы, визуализация).
- `experiment_results/` – папка с результатами:
  - 3 таблицы с исходными данными (CSV);
  - 4 графика (PNG);
  - таблица управления согласием.

## Запуск
1. Установите Python 3.11+.
2. Установите зависимости: `pip install pandas numpy matplotlib seaborn`
3. Запустите скрипт: `python experiment.py`

## Результаты
После выполнения в консоль выводится количество пользователей, найденных по запросу тренера (3). В папке `experiment_results` появляются все таблицы и графики.
