# Экспериментальная часть ВКР: Мультибазовая система для интеграции данных о здоровье

## Описание проекта
Данный репозиторий содержит исходный код и результаты экспериментов для магистерской выпускной квалификационной работы по теме:  
**«Построение мультибазовой системы, интегрирующей информацию о показателях здоровья пользователей тренажёрных залов»**.

Цель экспериментов — демонстрация работоспособности предложенной архитектуры медиатора, интегрирующего гетерогенные источники данных (реляционная БД, API фитнес-трекера, файловое хранилище активностей) и выполняющего поиск пользователей по биометрическим критериям с учётом статуса открытости к предложениям.

## Структура репозитория
- `experiment.py` — основной скрипт на Python:
  - генерация синтетических данных (500 пользователей);
  - объединение данных в глобальную виртуальную схему (медиатор);
  - выполнение поискового запроса тренера;
  - построение 4 графиков и сохранение таблиц.
- Сгенерированные файлы (все находятся в корне репозитория):
  - `table1_club_members.csv` — данные из реляционной БД (возраст, пол, посещаемость);
  - `table2_health_metrics.csv` — показатели здоровья (HRV, пульс, сон);
  - `table3_activities.csv` — детальные записи тренировок;
  - `user_openness.csv` — статус открытости и настройки видимости данных;
  - `fig1_hrv_distribution.png` — распределение HRV по статусу открытости;
  - `fig2_visits_by_age.png` — посещаемость по возрастным группам;
  - `fig3_sleep_hrv_corr.png` — корреляция сна и HRV;
  - `fig4_top_active_users.png` — топ-5 пользователей по количеству тренировок.

## Требования
- Python 3.11 или новее
- Установленные библиотеки: `pandas`, `numpy`, `matplotlib`, `seaborn`

   ```bash

4. Запустите скрипт:
